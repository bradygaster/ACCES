## Learnings

- First session: team initialized 2026-03-05
- Full TypeScript scaffold created: package.json, tsconfig.json, 8 source files under src/
- Dependencies: tsx, typescript, @types/node (dev), rss-parser, @octokit/rest (runtime)
- `npm run build` compiles clean with strict mode, ES2022 target, NodeNext resolution
- RSS feed discovery is fully implemented (rss-parser); GitHub search implemented (@octokit/rest)
- YouTube, Reddit, social media, and blog search are skeletons for scouts to fill
- State persistence uses JSON serialization of Set/Map via `.state/run-state.json`
- All 9 output files from ACCES.md §2 are generated by src/output.ts
- Pipeline flow: loadState → discoverContent + discoverCommunity → deduplicate → classify → analyze → generateOutput → saveState
- Canonical ID = sha256(title|url|author|date).slice(0,16)
- The `Channel` type uses `(string & {})` pattern to allow known unions + arbitrary strings
